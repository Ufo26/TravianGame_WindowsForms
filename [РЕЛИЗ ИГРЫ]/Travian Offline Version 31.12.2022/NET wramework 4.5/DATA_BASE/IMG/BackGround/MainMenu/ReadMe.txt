Папка "MainMenu" хранит фоны главного меню.
Картинка растягивается под текущие размеры клиента.
Программа генерирует случайную картинку из этой папки.
Вы можете добавлять любое количество картинок любого формата (.jpg, .png, .bmp, gif и т.д.),
с любым названием, но файл ReadMe.txt должен следовать первым в алфавитном порядке.
Количество картинок в этой папке должно быть в диапазоне [1..Бесконечность]
============================================================
The "MainMenu" folder stores the backgrounds of the main menu.
The picture is stretched to fit the current size of the client.

The program generates a random image from this folder.

You can add any number of images of any format (.jpg, .png, .bmp, gif, etc.),
with any name, but the file ReadMe.txt must follow first in alphabetical order.
The number of images in this folder should be in the range [1..Infinity]
